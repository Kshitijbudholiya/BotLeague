# BotLeague

> **Note: As per the project requirements, the backend has been removed. However, due to time constraints, some API integrations could not be fully removed or refactored. As a result, certain features such as login, registration, and other backend-dependent functionalities may not work as expected and can generate errors. Additionally, some console errors may still be present due to the remaining API calls.**

**BotLeague** is India's national robotics competition platform, designed to connect builders, teams, organizers, judges, sponsors, and robotics enthusiasts through a unified digital ecosystem.

From regional tournaments and live events to rankings, leaderboards, and community participation, BotLeague provides a centralized platform for the entire robotics competition landscape.

Built with **React**, **Flask**, **JWT Authentication**, and a modern responsive UI optimized for desktop and mobile devices.

---

## What Makes BotLeague Different

Most robotics competitions operate independently with fragmented registrations, rankings, and event systems.

BotLeague creates a unified national ecosystem where:

* Teams can build a public competitive profile
* Organizers can host and promote competitions
* Participants can track rankings across events
* Judges and volunteers can join verified programs
* Sponsors can connect directly with robotics communities
* Robotics achievements become portable across competitions

## Quick Start

### Clone Repository

```bash
git clone https://github.com/Kshitijbudholiya/BotLeague.git
cd BotLeague
```

---

## Frontend Setup

Install packages:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Frontend runs at:

```text
http://localhost:5173
```
